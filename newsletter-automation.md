# Automatisation Newsletter — MLM Décrypté
> Plateforme recommandée : **Brevo** (ex-Sendinblue) — Gratuit jusqu'à 300 emails/jour
> Mis à jour le 25 mars 2026

---

## Pourquoi Brevo ?

- Gratuit jusqu'à 300 emails/jour (largement suffisant pour démarrer)
- Interface en français
- Automatisation visuelle (drag & drop)
- Conforme RGPD (hébergement européen)
- Intégration directe dans le code HTML du site (API déjà intégrée dans article-template.html)

---

## CHECKLIST DE MISE EN PLACE SUR BREVO

### Étape 1 — Créer le compte Brevo
- Aller sur brevo.com → Créer un compte gratuit
- Vérifier votre domaine email (important pour la délivrabilité)

### Étape 2 — Créer la liste de contacts
- Contacts → Listes → Créer une liste "MLM Décrypté — Abonnés"
- Noter l'ID de la liste (à mettre dans le code HTML du site)

### Étape 3 — Récupérer la clé API
- Mon compte → Clés API → Créer une clé
- Coller la clé dans article-template.html (ligne : 'api-key': 'VOTRE_CLE_API_BREVO')

### Étape 4 — Créer les emails de la Séquence 1
- Automatisations → Nouvelle automatisation
- Déclencheur : "Contact ajouté à la liste"
- Ajouter les 3 emails (J0, J3, J7) avec les contenus ci-dessus

### Étape 5 — Créer les emails de la Séquence 2
- Faire suite à la Séquence 1 (même automatisation ou nouvelle)
- Ajouter les emails J+8, J+13, J+18, J+23, J+28

### Étape 6 — Créer l'automatisation de réengagement
- Déclencheur : "N'a pas ouvert d'email depuis 60 jours"
- Envoyer l'email de réengagement
- Si pas d'ouverture 7 jours après → Désabonner automatiquement

### Étape 7 — Tester avant de lancer
- S'abonner soi-même avec une adresse test
- Vérifier la réception de chaque email
- Vérifier les liens dans chaque email

## IDÉES DE LEAD MAGNETS (pour accélérer les inscriptions)

Un lead magnet est un contenu offert gratuitement en échange d'un email.

1. **"Guide : 7 questions à poser avant de rejoindre un MLM"** — PDF 2 pages, pratique, protège le lecteur
2. **"Checklist : Comment évaluer la légitimité d'une société MLM"** — 1 page, très partageable
3. **"Mini-formation email : Comprendre le MLM en 5 jours"** — Séquence de 5 emails courts, haute valeur perçue

*Le lead magnet n°1 est le plus simple à créer et le plus efficace pour votre positionnement.*
